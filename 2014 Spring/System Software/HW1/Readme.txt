This program will compile from gcc and takes the code input file as an argument. SIO output will be printed at the bottom of the program. If you wish to save this to a text file please use Bash's ">" operator.

To compile:
gcc -o PL0VM.bin VM_main.c

To execute:
./PL0VM.bin <YOUR CODE FILE>

Refer to test output.txt for the sample program and output.
